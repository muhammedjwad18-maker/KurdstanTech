## 🚀 چۆن بە کاری بهێنیت

وێبسایتەکە **یەک فایلی HTML** یە — پێویستی بە هیچ فریمووەرک یان سێرڤەری تایبەتی نییە.

### بنیشتنی ئاسان (Local)
```
فایلی kurdstab-tech.html کلیک دوبارە بکە
```
بە ئاوتۆماتیکی لە براوزەرەکەت دەکرێتەوە.

### هۆستنگ
فایلەکە ئاماد‌ەی ئەپلۆد کردنە لەسەر:
- **Netlify** — فایل بڕیشکێنه‌رەوه، ئاوتۆماتیک دەردەچێت
- **Vercel** — هەمان شێوازی Netlify
- **cPanel Hosting** — فایل بکێشه‌ بۆ `public_html`
- **GitHub Pages** — ڕیپۆ دروست بکە، فایل بخە، GitHub Pages چالاک بکە

### گۆڕینی ژمارەی تەلەفۆن
لەناو `kurdstab-tech.html` بگەڕێ بۆ:
```
9647700998243
```
هەموو دیارکردنەکانی بگۆڕە بە ژمارەی نوێ.

### گۆڕینی ئیمەیل
بگەڕێ بۆ:
```
info@kurdstabtech.com
```

### زیادکردنی لینکی سۆشیال میدیا
لەناو فایلەکە بگەڕێ بۆ:
```html
<a href="#" target="_blank" class="social-link" title="فەیسبووک">📘</a>
```
`#` بگۆڕە بە لینکی ڕاستەقینە، بۆ نموونە:
```html
<a href="https://facebook.com/KurdstabTech" target="_blank" ...>
```

### گۆڕینی نرخەکان
بگەڕێ بۆ `class="plan-price"` و نرخەکان بگۆڕە.

### زیادکردنی وێنە / لۆگۆ
لە بەشی `logo-icon` وێنەکەت دابنێ:
```html
<img src="logo.png" alt="KS Tech" width="42" height="42" />
```
